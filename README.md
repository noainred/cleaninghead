# BrainBloom 🌱

> 브라우저에서 바로 쓰는 개인용 마인드맵 · 브레인스토밍 도구

**👉 [www.redmir.net](https://www.redmir.net) 에 접속하면 바로 사용할 수 있습니다 — 설치도 가입도 필요 없습니다.**

서버 없이 단일 HTML 파일로 동작하는 가벼운 도구입니다. 왼쪽에 텍스트로 적으면 다이어그램이 자동으로 그려지고, 다이어그램에서 편집하면 텍스트가 자동으로 갱신됩니다. 회의 정리·글 구조 잡기·할 일 쪼개기·브레인스토밍 등 "생각을 펼치고 묶는" 모든 일에 쓸 수 있습니다.

---

## ✨ 주요 기능

### 입력 & 편집
- **양방향 동기화** — 왼쪽 텍스트와 가운데 다이어그램이 실시간으로 서로를 반영
- **들여쓰기 표기** — 들여쓰기로 계층 표현, 또는 `AA-A` 하이픈 표기
- **키보드 중심 편집** — `Tab`(자식)·`Enter`(형제)로 노드를 즉시 추가하고 바로 입력
- **드래그 앤 드롭** — 노드를 끌어 부모 변경, `Alt`+방향키로 순서·계층 이동
- **검색 · 접기/펼치기** — `Ctrl+F` 검색, `Space`로 가지 접기/펼치기

### 시각화 & 꾸미기
- **아웃라인 뷰** — 헤더 🗂 버튼이나 `F` 키로 마인드맵 ↔ 아웃라인(들여쓰기 목록) 전환. **깊이별 색 블록**으로 계층을 한눈에, 선택 항목이 속한 그룹 강조, 행에서 바로 편집·`Enter`로 확정·접기/펼치기·할 일 체크. ↑/↓로 목록 이동(그룹 경계는 두 번), →펼치기·←접기. 맵으로 돌아가면 선택 노드를 중앙에 표시
- **간트 차트 · 타임라인 뷰** — 헤더 📊/🕒 버튼이나 `G`·`T` 키(우클릭 메뉴)로 일정 중심의 두 뷰로 전환. 우측 패널 **"일정"** 에 **시작일·종료일·마일스톤**을 넣으면 간트 막대로 보이고, **막대를 끌어 일정 이동·우측 끝을 끌어 기간 조절**. 부모는 하위를 합친 **요약 막대(롤업)**, 진행률은 막대 채움, 관계선은 **의존성 화살표**, 빨간 선은 오늘. **타임라인**은 상위 가지별 스윔레인에 날짜 점(◆ 마일스톤)으로 가볍게 표시
- **14색 팔레트** — 카테고리별 자동 배색, 노드 아이콘(이모지), **노드 모양**(둥근·각짐·알약·타원), 메타데이터(날짜·작업량·비용)
- **할 일 · 진행률** — 노드를 체크박스 "할 일"로 만들고, 하위 완료 개수(예: 2/4)를 자동 표시
- **노드 크기** — 라벨 길이에 맞춰 자동 조정(한글 폭 추정) + **우측 핸들을 드래그해 폭을 직접 조절**(더블클릭하면 자동 폭으로), 노드 줄 수·간격(촘촘/보통/넓게)·폰트 설정
- **배경 테마 6종** — 밝은 3종(샌드·세이지·스카이) + 어두운 3종(차콜·미드나잇·포레스트) + 점 패턴
- **줌 & 자동 맞춤** — `Ctrl`+휠 확대, 캔버스 어디서든 우클릭 메뉴로 배율·보기 모드 조절. 처음 열 때·창 크기 변경 시 화면에 자동 맞춤
- **몰입 모드 · 집중 모드** — 발표·집중용 보기 전용 모드와 편집 중 관련 줄기만 강조(흐림 3단계)
- **하위 일괄 색상** — 자식이 있는 노드의 색을 바꿀 때 "하위 전체 / 이 노드만"을 물어, 가지 전체를 한 번에 통일
- **속성 패널 · 노드 세부 정보** — 노드 우클릭 → "노드 세부 정보"로 우측 패널 열기, 패널 위·아래 "패널 닫기"나 빈 바탕 클릭으로 닫기
- **업데이트 알림** — 새 버전 접속 시 이번 변경점을 화면을 가리지 않는 카드로 1회 안내(설정에서 끄기)

### 연결 · 묶기 · 필터
- **관계선(크로스링크)** — 부모-자식이 아닌 두 노드를 화살표로 잇기. **곡선·직선·직각** 중 선택, 점 3개를 끌어 곡률 조절, **우클릭**으로 점선/실선·두께 변경. 선을 고르면 노드 위로 떠올라 가려지지 않음
- **그룹 · 경계** — 한 노드의 하위 전체를 둥근 박스로 묶는 **경계**, 떨어진 여러 노드를 **Shift+클릭**으로 자유롭게 묶는 **그룹**. 통계에 그룹·경계 개수 표시
- **아이콘 몰아보기 · 태그 필터** — 화면 하단 바에 지도에 쓰인 아이콘·태그를 모아 보고, 클릭하면 해당 노드만 또렷하게 필터링. 아이콘은 사용자가 정한 줄로 분류 가능

### 집중 도구
- **타이머** — 정한 시간 동안 집중. 시작·경과·3·2·1분 전·종료를 화면 중앙 알림으로 표시
- **첫 실행 온보딩** — 처음 오면 단순한 화면 + 시작 가지 + 격언으로 부담 없이 시작
- **오늘의 단어 · 오늘의 격언** — 발상을 돕는 문구(설정에서 편집 가능)

### 저장 & 내보내기
- **여러 문서 관리** — 헤더 **📄 문서** 버튼으로 문서 목록을 열어 **새 문서 만들기 · 전환 · 이름 변경 · 삭제**. 각 문서는 마인드맵·텍스트를 따로 보관하고, 새로고침하면 마지막에 열어 둔 문서가 복원됨(이름은 루트 노드 제목을 따라 자동, 직접 변경 가능)
- **자동 백업** — IndexedDB에 자동 저장, 창을 닫았다 열어도 이어서 작업
- **새로고침해도 화면 그대로** — 배율·스크롤 위치·접힌 가지·선택 노드를 1초마다 기억해, 새로고침하면 떠날 때 모습 그대로 복원
- **내보내기 6종** — PDF(한글 안전) · JPG · SVG · CSV · MD · JSON. 파일명은 `문서이름_시각.확장자`로 저장돼 여러 문서를 내보내도 구분 가능
- **JSON 백업 열기** — 내보낸 JSON을 다시 불러오기

### 구글 드라이브 동기화 (다른 기기에서 이어 쓰기)
- 앱 전용 폴더에 **날짜별 버전**으로 자동/수동 저장 (권한은 `drive.file` — 앱이 만든 파일만 접근)
- **다중 기기 충돌 시 저장본 비교 화면** — 현재 화면과 드라이브 최신본을 **좌우 아웃라인으로 펼쳐** 차이(🟢추가·🔴삭제·🟡변경)를 색으로 비교한 뒤 **불러오기 / 합치기(Merge) / 무시 / 파일 삭제** 중 선택. 설정 백업·복원
- 파일 이름 접두어는 **브라우저마다 고정 배정**돼 다른 기기·브라우저 저장본과 섞이지 않음

### 연동
- **📅 구글 캘린더** — 마인드맵을 일정으로 추가
- **🤖 AI 요약** — 마인드맵을 클립보드에 복사하고 ChatGPT · Claude · Gemini 채팅을 새 탭으로 열기

## ⌨️ 단축키

| 키 | 동작 |
|---|---|
| **Tab** | 자식 노드 추가 + 즉시 편집 |
| **Enter** / **Shift+Enter** | 형제 노드 추가(아래/위) + 즉시 편집 |
| **F2** / 더블클릭 | 라벨 편집 |
| **← → ↑ ↓** | 노드 사이 이동 |
| **Alt + 방향키** | 노드 순서·계층 이동 |
| **Space** | 가지 접기 / 펼치기 |
| **Shift + ←** / **→** | 고른 가지 접기 / 펼치기 (평범한 ←/→는 노드 이동) |
| **F** | 마인드맵 ↔ 아웃라인 화면 전환 |
| **G** / **T** | 간트 / 타임라인 화면 전환 |
| **Delete** / Backspace | 노드 삭제 |
| **Esc** | 편집 취소 / 선택 해제 |
| **Ctrl/⌘ + Z** / **Ctrl/⌘ + Shift + Z** | 실행 취소 / 다시 실행 |
| **Ctrl + F** | 노드 검색 |
| **Ctrl + Shift + S** | 구글 드라이브에 저장 |
| **Ctrl + Shift + C** | 구글 캘린더에 추가 |
| **+** / **−** | 확대 / 축소 |
| **Ctrl + 휠** | 확대 / 축소 |
| 우클릭(어디서나) | 배율 · 편집/몰입 모드 (노드 위: 노드 세부 정보) |

> 맥(Mac)에서는 `Ctrl` 대신 `⌘`(Command).

## 🔒 개인정보 보호

- 마인드맵 내용은 **본인 브라우저(IndexedDB)** 와 — 원할 때만 — **본인 구글 드라이브**에만 저장됩니다.
- 마인드맵 내용은 **서버로 전송되지 않습니다.** 구글 드라이브 연동도 앱이 만든 파일(`drive.file`)에만 접근합니다.
- 방문 통계(Google Analytics)는 공식 사이트(redmir.net)에서만 수집되며, 마인드맵 내용과는 무관합니다.
- 협업 기능이 없어 내가 만든 마인드맵을 다른 사람이 볼 수 없습니다.

## 🛠 기술 스택

- **React 18 + Babel Standalone** — 브라우저에서 JSX를 즉석 컴파일(빌드 도구 없음)
- **IndexedDB** — 로컬 자동 저장 / **Google Drive API + Google Identity Services** — 선택적 동기화
- **jsPDF** — PDF 출력 / CDN 리소스는 **SRI 무결성 해시**로 고정
- **단일 HTML 파일 · 외부 백엔드 없음** — 인터넷 연결만 있으면 동작
