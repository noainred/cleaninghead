# BrainBloom 🌱

> 브라우저에서 바로 쓰는 한국어 마인드맵 / 브레인스토밍 도구

서버 없이 단일 HTML 파일로 동작하는 가벼운 마인드맵 도구입니다.
텍스트로 입력하면 다이어그램이 자동으로 그려지고, 다이어그램에서 편집하면
텍스트가 자동으로 갱신됩니다.

## ✨ 주요 기능

### 입력 & 편집
- **양방향 동기화** — 텍스트와 다이어그램이 자동으로 연결됨
- **들여쓰기 표기** — 2칸 들여쓰기로 계층 표현, 또는 `AA-A` 하이픈 표기
- **인라인 노드 편집** — Tab/Enter로 자식·형제 노드 즉시 추가 + 바로 입력
- **드래그 앤 드롭** — 노드를 끌어서 부모 변경
- **자동 들여쓰기** — 들여쓰기된 줄에서 Enter 시 같은 들여쓰기 유지

### 시각화 & 꾸미기
- **노드 색상** — 14색 팔레트, 카테고리별 자동 배색
- **노드 아이콘** — 7개 카테고리 × 이모지 (우선순위 1~5, 상태, 중요도, 감정, 분류, 기호, 활동)
  - 카테고리당 1개씩 선택 가능 (총 7개까지 조합)
- **메타데이터** — 날짜·작업량·비용 표시
- **가변 노드 크기** — 라벨 길이에 따라 자동 조정, 한글 폭 추정
- **줌 & 패닝** — Ctrl+휠 확대, 빈 공간 드래그로 이동

### 저장 & 내보내기
- **5가지 형식** — PDF · JPG · SVG · CSV · JSON
- **한글 안전** — PDF는 SVG→PNG 방식으로 변환해 한글 깨짐 없음
- **자동 백업** — IndexedDB에 다이어그램 자동 저장, 다음 날 이어서 작업 가능

### 추가 기능
- **📅 구글 캘린더 연동** — 마인드맵을 마크다운 본문으로 변환해 오늘 종일 일정으로 등록
  - 일정 제목은 설정에서 자유롭게 변경 가능
- **오늘의 격언** — 200개 위인 명언 + 사용자 격언 추가 가능. 매일 자동 순환
- **Undo/Redo** — Ctrl+Z, Ctrl+Shift+Z, 최대 100단계
- **새로 시작** — 깨끗한 빈 다이어그램 시작 (Ctrl+Z로 복구 가능)

## ⌨️ 단축키

| 키 | 동작 |
|---|---|
| **Tab** | 자식 노드 추가 + 즉시 편집 |
| **Enter** | 형제 노드 추가 + 즉시 편집 |
| **← → ↑ ↓** | 노드 사이 이동 |
| **F2** / 더블클릭 | 라벨 편집 |
| **Delete** / Backspace | 노드 삭제 |
| **Alt + ↑↓** | 노드 순서 이동 |
| **Esc** | 선택 해제 / 새 노드 취소 |
| **Ctrl/⌘ + Z** | 실행 취소 |
| **Ctrl/⌘ + Shift + Z** | 다시 실행 |
| **Ctrl + 휠** | 확대 / 축소 |
| 빈 공간 드래그 | 화면 이동 |

## 🚀 사용 방법

### 방법 1: 로컬에서 바로 사용
1. `brainstorm.html` 다운로드
2. 브라우저로 더블클릭해서 열기 — 끝

### 방법 2: GitHub Pages로 호스팅
1. GitHub 저장소 만들기 (Public)
2. `index.html` 업로드 (`brainstorm.html`과 내용 동일, 이름만 다름)
3. Settings → Pages → Source: `main` branch / root 선택
4. `https://사용자명.github.io/저장소이름/` 으로 접속

## 🛠 기술 스택

- **React 18** — UI (브라우저에서 Babel로 in-place 컴파일)
- **jsPDF** — PDF 출력
- **IndexedDB** — 로컬 자동 저장
- **외부 백엔드 없음** — 단일 HTML 파일로 완결

### CDN 의존성
- React 18 (`unpkg.com/react`)
- Babel Standalone (`unpkg.com/@babel/standalone`)
- jsPDF (`unpkg.com/jspdf`)
- Google Fonts (Fraunces, Plus Jakarta Sans, JetBrains Mono)

인터넷 연결이 필요합니다. 오프라인 사용은 향후 검토 중입니다.

## 🔒 개인정보 보호

- 모든 데이터는 **본인 브라우저(IndexedDB)에만 저장**됩니다
- 서버로 전송되는 정보 없음
- 구글 캘린더 연동도 OAuth 없이 URL 링크 방식 — 사용자가 직접 캘린더에서 저장
- 누가 만든 마인드맵을 다른 사람이 볼 수 없습니다 (협업 기능 없음)
