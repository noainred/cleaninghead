1BrainBloom이란?

머릿속 생각을 가운데서 바깥으로 뻗어 나가는 가지 모양으로 정리하는 도구입니다.

하나의 중심 생각에서 시작해, 떠오르는 아이디어를 가지처럼 계속 붙여 나갑니다. 회의 정리, 글의 구조 잡기, 할 일 쪼개기, 브레인스토밍 등 "생각을 펼치고 묶는" 모든 일에 쓸 수 있습니다.

처음이라도 괜찮아요 — 이 도구가 좋은 5가지 이유
  • 열면 바로 시작 — 설치도 가입도 없습니다. 브라우저로 이 페이지를 여는 게 전부고, 무료입니다.
  • 마인드맵을 그릴 줄 몰라도 됩니다 — 왼쪽에 메모하듯 적고 들여쓰기만 하면 지도가 자동으로 그려집니다. 반대로 그림을 만지면 글로도 돌아옵니다.
  • 생각의 속도를 손이 따라갑니다 — Tab이면 가지, Enter면 형제 생각. 키보드만으로 쭉쭉 적을 수 있어, 도구를 만지느라 떠오른 생각이 날아가지 않습니다.
  • 저장 걱정이 없습니다 — 적는 족족 알아서 보관되고, 창을 닫았다 열어도 이어집니다. 원하면 구글 드라이브에 날짜별 버전으로 자동 백업할 수 있습니다.
  • 막막할 때 시작을 도와줍니다 — 오늘의 단어와 격언이 발상을 툭 건드려 주고, 타이머로 3분만 몰입해 볼 수 있고, 집중·몰입 모드가 한 줄기만 또렷하게 잡아 줍니다.

크롬(Chrome) 또는 엣지(Edge) 브라우저에서 가장 잘 작동합니다. 적은 내용은 어느 서버에도 올라가지 않고, 내 브라우저와 — 원할 때만 — 내 구글 드라이브에만 저장됩니다.

2바로 시작하기

3분이면 첫 마인드맵을 만들 수 있습니다.

  1. 브라우저로 BrainBloom 페이지에 접속합니다.
  2. 화면 왼쪽의 입력창에 첫 번째 생각(중심 주제)을 적습니다. 그러면 가운데에 첫 노드가 생깁니다.
    ※ '노드'란 마인드맵의 동그란 칸 하나하나를 말합니다.
  3. 이어서 떠오르는 생각을 줄을 바꿔 가며 적으면, 가지가 자동으로 그려집니다.
  4. 마음에 드는 모양이 되면 저장합니다(7번 항목 참고).
💡
처음엔 예시가 들어 있을 수 있습니다. 설정 → 새로 시작으로 깨끗한 화면에서 시작할 수 있고, 실수로 지웠어도 Ctrl + Z로 되돌릴 수 있으니 마음껏 시도해 보세요.

3생각을 적는 두 가지 방법

키보드로 빠르게 적거나, 마우스로 직접 만들 수 있습니다.

방법 1. 왼쪽 입력창에 글로 적기

왼쪽 입력창에 생각을 줄 단위로 적습니다. 들여쓰기(띄어쓰기/탭)로 가지의 깊이를 표현합니다. 예를 들어:

여행 준비
    항공권 예약
    숙소 알아보기
        호텔 후기 확인
    짐 싸기

이렇게 적으면 '여행 준비'를 중심으로 그 아래에 가지들이 자동으로 그려집니다. 글을 고치면 그림도 바로 따라 바뀝니다.

방법 2. 노드를 직접 클릭하며 만들기

가운데 그림에서 노드를 클릭해 고른 뒤, 키보드 단축키로 새 가지를 빠르게 추가할 수 있습니다. 이 방법이 익숙해지면 가장 빠릅니다. 단축키는 다음 항목을 보세요.

4단축키 한눈에

노드를 하나 클릭해 고른 상태에서 아래 키를 누르면 됩니다.

📱
휴대폰·태블릿에서도 써요. 한 손가락으로 빈 곳을 끌면 화면이 움직이고, 노드를 두 번 톡톡 두드리면 편집할 수 있어요. 자식 추가(＋)·삭제(×) 버튼은 터치 화면에서 항상 보입니다. (새 가지를 많이 만드는 작업은 키보드가 있는 컴퓨터가 더 편해요.)

5노드 꾸미기 (색 · 아이콘 · 정보)

노드에 색과 아이콘을 넣어 한눈에 구분되게 만들 수 있습니다.

  • 색상 — 가지마다 다른 색을 주어 분류할 수 있습니다. 새 가지를 만들면 색이 자동으로 어울리게 입혀집니다.
  • 아이콘 — 노드에 작은 그림(이모지)을 붙여 의미를 빠르게 표시할 수 있습니다.
  • 추가 정보 — 노드에 날짜, 노력(중요도), 비용 같은 정보를 함께 적어 둘 수 있습니다.
  • 링크 — 글에 인터넷 주소를 적으면 클릭할 수 있는 링크가 됩니다. [이름](주소) 형식으로 적으면 주소 대신 '이름'이 보이게 할 수도 있습니다.
💡
노드를 골라 두면 관련 꾸미기 도구가 나타납니다. 이것저것 눌러 보세요 — 무엇을 하든 Ctrl + Z로 되돌릴 수 있습니다.

6화면 다루기 (확대 · 접기 · 검색 · 몰입)

생각이 많아져도 화면을 편하게 둘러볼 수 있습니다.

  • 확대 / 축소 — Ctrl을 누른 채 마우스 휠을 굴리면 가까이/멀리 볼 수 있습니다.
  • 접기 / 펼치기 — 가지가 복잡하면 Space로 접어서 깔끔하게 볼 수 있습니다. 도구 모음의 버튼으로 한 번에 전체를 접거나 펼칠 수도 있습니다.
  • 검색 — Ctrl + F로 원하는 단어가 든 노드를 찾습니다. 찾은 노드가 잠깐 반짝여 위치를 알려 줍니다.
  • 처음 위치로 — 멀리 스크롤하다 길을 잃었다면, 노드가 없는 빈 곳을 더블클릭하세요. 가장 위 노드(루트)가 화면 가운데로 돌아옵니다.
  • 타이머 — 정해진 시간 동안 집중하고 싶을 때 타이머를 쓸 수 있습니다(기본 3분, 설정에서 조절).

몰입 모드 — 발표하거나 집중해서 볼 때

빈 곳을 마우스 우클릭하면 편집 모드 / 몰입 모드를 고를 수 있습니다.

  • 편집 모드(기본) — 평소처럼 자유롭게 만들고 고치는 상태입니다.
  • 몰입 모드 — 편집이 잠긴 보기 전용 상태입니다. 노드를 클릭하면 그 노드까지의 줄기와 바로 아래 자식만 또렷하게 보이고, 나머지는 흐려집니다. 발표하거나 한 줄기에 집중해 살펴볼 때 좋습니다.
  • 몰입 모드 중에는 화면 아래에 🎯 몰입 모드 표시가 떠 있고, 그걸 누르면 편집 모드로 돌아옵니다.
💡
편집할 때도 집중 효과를 쓸 수 있어요. 설정 → 화면 표시에서 "편집 중 집중 모드"를 켜면, 노드를 편집하거나 추가하는 동안 관련 줄기만 또렷하게 보입니다. 흐림 정도(약하게/보통/강하게)도 고를 수 있습니다.

7저장과 내보내기

작업은 자동으로 보관되며, 여러 형식으로 꺼낼 수 있습니다.

자동 보관 (이 컴퓨터 안에)

작업 내용은 이 브라우저 안에 자동으로 보관됩니다. 실수로 창을 닫아도, 다시 열면 이어서 작업할 수 있습니다. 단, 이건 '이 컴퓨터/이 브라우저' 안에서만입니다. 다른 컴퓨터에서도 보려면 아래 구글 드라이브 연동을 쓰세요.

파일로 내보내기 · 다시 열기

만든 마인드맵을 다음 형식으로 내보낼 수 있습니다:

  • 이미지(JPG) / 그림(SVG) — 그림 형태로 저장
  • PDF — 문서로 저장·인쇄
  • 표(CSV) — 엑셀 등에서 열기
  • 데이터(JSON) — 나중에 다시 불러올 수 있는 원본 형태
  • 마크다운(MD) — 글 문서 형태
📂
JSON 백업은 다시 열 수 있어요. 설정의 "📂 JSON 백업 파일 열기" 버튼으로 내 컴퓨터의 JSON 파일을 불러옵니다. 브라우저를 청소해도, 컴퓨터를 바꿔도 살아남는 가장 확실한 백업이에요. 열면 현재 작업이 바뀌지만 Ctrl+Z로 되돌릴 수 있습니다.

8구글 드라이브 연동 — 다른 기기에서 이어 쓰기

회사에서 작업한 것을 집에서 이어서 하고 싶을 때 쓰는 핵심 기능입니다.

처음 연결하기

  1. 화면 위쪽의 설정(⚙)을 엽니다.
  2. 맨 아래 구글 드라이브 연동에서 "구글 드라이브에 연결"을 누릅니다.
  3. 구글 로그인 창이 뜨면, 사용할 구글 계정으로 로그인하고 권한을 허용합니다.
  4. 연결되면 "지금 저장"으로 저장하거나, 자동저장을 켜 둘 수 있습니다.
🔒
이 앱은 자신이 만든 파일만 다룹니다. 구글 드라이브의 다른 파일은 보지 않으니 안심하세요. 저장 파일은 드라이브의 BrainBloom 폴더에 모입니다.

자동저장

자동저장을 켜면 정한 간격(예: 1분)마다 알아서 저장합니다. 내용이 바뀌었을 때만 저장하므로, 가만히 두면 같은 내용이 쌓이지 않습니다. 하루에 보관할 버전 개수는 설정의 "오늘 버전 보관 개수"에서 1~100개로 정할 수 있고(기본 5개, 수동 저장에도 같은 규칙 적용), 날짜가 바뀌면 전날 것은 가장 최신 1개만 남겨 정리합니다.

다른 컴퓨터에서 불러오기

  1. 다른 컴퓨터에서 같은 페이지에 접속합니다.
  2. 설정 → 구글 드라이브 연동에서 같은 구글 계정으로 연결합니다.
  3. "불러오기 목록"을 누르면 저장된 파일이 최신순으로 보입니다(수정 시각·크기 표시). 파일을 누르면 미리보기가 떠서 노드 수·중심 주제·내용 앞부분을 확인할 수 있고, "⬇ 이 파일 불러오기"를 눌러야 실제로 불러옵니다.
⚙️
앱 설정도 가져갈 수 있어요. 드라이브 연동 칸의 "⚙️⬆ 설정 저장"을 눌러 두면, 다른 컴퓨터에서 "⚙️⬇ 설정 불러오기"로 배경 색·버튼 구성 같은 설정을 그대로 가져옵니다. 버튼을 누를 때만 저장·적용됩니다(자동 아님).

"설정" 글자 색의 의미

자동저장을 켜 두면, 위쪽 "설정" 글자 색으로 상태를 알려 줍니다. 색이 헷갈리면 설정 버튼에 마우스를 올려 보세요 — 지금 어떤 상태인지 설명이 바로 뜹니다.

  • 파란색 — 자동저장이 잘 되고 있습니다. (안심)
  • 빨간색 — 연결이 끊겼거나 저장에 실패했습니다. 설정을 눌러 다시 연결하세요.
  • 평소 색 — 자동저장을 쓰지 않는 상태입니다.
💡
다른 컴퓨터에서 더 최근에 저장한 내용이 있으면, "다른 기기에서 작업한 최신본이 있어요" 창이 뜹니다. 창에는 지금 화면과 최신본의 차이(노드 수, 최신본에만 있는 가지 등)와 최신본 내용 미리보기가 자동으로 표시되고, 세 가지 중 고를 수 있어요 — ⬇ 최신본 불러오기(화면 교체), 🔀 합치기(지금 화면은 그대로 두고 중심 아래 "복원" 가지에 최신본 전체를 붙임), 무시하고 계속(지금 내용 유지). 어떤 선택이든 Ctrl+Z로 되돌릴 수 있고, 모든 저장은 별도 버전 파일로 남아 데이터가 사라지지 않습니다. 이 확인은 연결될 때·저장 직전·창 복귀 때·10분마다 자동으로 이뤄집니다.
⚠️
구글 드라이브 연동은 보안상 정해진 인터넷 주소에서만 작동합니다. 또한 일정 기간(약 7일)이 지나면 한 번씩 다시 연결해야 할 수 있습니다 — 이는 구글의 정책입니다.

9설정 살펴보기

설정(⚙)에서 화면과 동작을 내 취향에 맞게 바꿀 수 있습니다.

  • 화면 표시 — 좌우 패널, 안내 문구 등을 켜고 끕니다.
  • 배경 색 — 눈이 편한 배경을 고릅니다. 밝은 톤 3가지(샌드·세이지·스카이)와 어두운 화면 3가지(차콜·미드나잇·포레스트)가 있고, 배경의 점 무늬도 켜고 끌 수 있습니다. 흰 화면이 오래 보기 피로하다면 바꿔 보세요.
  • 편집 중 집중 모드 — 노드를 편집할 때 관련 줄기만 또렷하게 보이는 기능과 흐림 정도를 정합니다.
  • 중복 탭 알림 — 같은 브라우저에서 BrainBloom을 두 탭에 동시에 열면 경고를 띄울지 정합니다. (두 탭에서 같이 편집하면 한쪽 작업이 덮어써질 수 있어서요.)
  • 노드 길이 · 폰트 — 글자 줄 수, 글꼴을 조절합니다.
  • 타이머 — 집중 시간(분)을 정합니다.
  • 시작 시 동작 — 앱을 열 때 지난 작업을 이어 받을지, 새로 시작할지 정합니다.
  • 오늘의 격언 · 새 단어 — 새 작업을 시작할 때 보여 주는 문구를 고릅니다.
  • 구글 드라이브 연동 — 위 8번 항목 참고.

설정의 각 항목에는 짧은 설명이 함께 적혀 있으니, 천천히 둘러보며 켜고 꺼 보세요.

10자주 묻는 질문 · 문제 해결

저장한 내용이 다른 컴퓨터에서 안 보여요

양쪽이 같은 구글 계정으로 연결됐는지 확인하세요. 계정이 다르면 파일이 보이지 않습니다. 그리고 다른 컴퓨터에서는 "불러오기 목록"을 눌러 직접 불러와야 합니다.

새로고침했더니 드라이브 연결이 풀렸어요

한 번 연결한 적이 있으면, 새로고침 후 잠깐 기다리면 자동으로 다시 연결됩니다. 그래도 안 되면(시간이 오래 지난 경우) 설정에서 "구글 드라이브에 연결"을 한 번 더 누르면 됩니다.

"설정" 글자가 빨개졌어요

자동저장이 끊겼다는 뜻입니다. 설정을 누르면 구글 드라이브 연동 항목으로 이동하고, 거기서 다시 연결하면 됩니다. (이 색 알림이 거슬리면 설정에서 끌 수도 있습니다.)

갑자기 노드 편집이 안 돼요

몰입 모드가 켜져 있는지 확인하세요. 화면 아래에 🎯 몰입 모드 표시가 있다면 그걸 클릭하면 편집 모드로 돌아옵니다. (빈 곳을 우클릭해서 바꿀 수도 있습니다.)

일부 노드가 흐리게 보여요

두 가지 경우입니다. 몰입 모드에서 노드를 클릭했거나, 편집 중 집중 모드가 켜진 상태에서 편집 중인 경우입니다. 몰입 모드라면 빈 곳을 클릭하면 흐림이 풀리고, 집중 모드는 편집을 끝내면 원래대로 돌아옵니다.

실수로 다 지웠어요

Ctrl + Z를 눌러 되돌리세요. 여러 번 눌러 더 이전으로도 갈 수 있습니다.

화면이 너무 크거나 작아요

Ctrl을 누른 채 마우스 휠을 굴려 확대·축소하세요.
